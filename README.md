## Nasıl Çalıştırılır

1. Install the required Node.js dependencies:
  ```bash
  npm install
  ```

2. Activating the virtual env:
  ```bash
  python -m venv .venv
  pip install -r requirements.txt
  ```

3. And run this command for the start/test:
  ```bash
  cd src
  python __main__.py
  ```
